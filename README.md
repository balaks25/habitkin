# Habitkin

Habitkin is an innovative app designed to help users build and maintain healthy habits through an engaging and user-friendly interface. This README provides an overview of the app's core features:

## Onboarding
During the onboarding process, users are guided through a series of steps to set up their accounts and personalize their experience. This includes:
- Creating an account
- Setting initial goals
- Customizing habit categories

## Home Tab
The Home tab serves as the main dashboard for users, providing:
- An overview of current habits and their progress
- Quick access to add new habits
- Motivational quotes and reminders

## Shop Tab
In the Shop tab, users can:
- Browse and purchase habit-related items or resources
- Unlock premium features and content with in-app currency
- Access community-generated content and ideas

## Progress Tab
The Progress tab allows users to:
- Track their habit completion rates over time
- View graphs and statistics for visual representation of progress
- Set reminders and notifications to stay on track

## Manage Tab with PIN Security
The Manage tab contains settings for users to:
- Edit or delete existing habits
- Configure app settings, notifications, and preferences
- Secure sensitive information with a PIN for added security

Habitkin aims to empower users to achieve their personal goals by combining functionality with a fun user experience!
